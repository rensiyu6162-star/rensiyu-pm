个人AI能力与落地应用全景

围绕视觉生成、AI产品功能落地、AI Coding与Agent提效、业务场景挖掘四个方向，形成从创意表达、需求定义、功能上线到效率提升的完整AI产品能力闭环。

PRODUCT x AI x DESIGN
01

AI视觉生成与设计落地

Prompt / Visual Style / Design Assets

熟练使用 Midjourney、可灵AI、即梦AI 等生图工具，能通过精细化 Prompt 控制视觉风格、亚文化表达、国风工艺质感与页面氛围。

理解国内外模型差异：Midjourney 更适合复杂视觉与 UI 装饰画面，国内模型在传统文化、国风内容与中文语境上更具适配度。

本科毕设《考工记》古籍工艺可视化项目中，提取工艺流程并批量构建 Prompt，生成一整套传统工艺插画素材。

02

AI产品项目实战

Online Feature / Model Logic / Compliance

完整参与网易 C 端 AI 相册记录项目，覆盖本地图片筛选模型、数据标注、云端声纹大模型 Prompt 调优与功能体验闭环。

设计 AI 自动聚合照片逻辑：剔除劣质、隐私、重复图片，生成 1-9 张事件卡片并自动配文，实现轻量化生活记录。

同步考量隐私与合规风险，区分本地/云端处理流程，设计用户授权弹窗，并在复盘中提出人像权重优化方案。

上线结果

功能上线后对渗透率与留存产生正向提升，也沉淀了从需求定义、模型边界判断到合规上线的AI产品实战经验。

03

AI提效工具与AI Coding实操

Prototype / Agent / Workflow

日常使用 Google AI Studio、Claude 等工具快速生成可交互产品原型页面，并结合逐轮校验修复处理 AI 幻觉、逻辑断裂与元素缺失。

利用内部 Skill / 自动化工具封装多语言翻译、网页自动走查等重复流程，将机械工作转化为可复用工作流，效率提升 30%+。

借助大模型辅助 PRD、需求文档与信息整理，把时间释放给业务判断、方案设计与体验细节决策。

独立完成《考工记》交互式可视化原型、个人作品集官网等可部署网页 Demo，并生成可访问 URL。

04

AI业务落地思维

Scenario / Value / Product Judgment

能主动在出行、营销、履约、内容记录等业务场景中挖掘 AI 落地机会，判断哪些环节适合自动化、生成式增强或 Agent 化。

清楚 AI 工具边界：AI 更适合标准化、重复性与生成辅助类任务，核心业务判断、用户需求权衡与风险取舍仍由产品主导。

结合交互设计专业背景，将 AI 生成内容与用户心智、页面场景、阅读路径结合，平衡效果、可用性与业务目标。

05

持续学习与趋势跟进

Agent / Vibe Coding / New Tools

持续关注 Agent、Vibe Coding、多模态生成与 AI 产品化趋势，优先把新工具转化为真实工作流，而不是停留在概念层。

在产品工作中保持"先判断场景价值，再选择AI能力"的方法，避免为了使用 AI 而使用 AI。

目标是把 AI 从单点工具变成产品推进的增效系统，服务于体验优化、业务数据提升与项目落地效率。
